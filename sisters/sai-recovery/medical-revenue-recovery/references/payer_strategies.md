# Payer-Specific Recovery Strategies

## Commercial Insurance Payers

### Aetna
**Negotiation Approach:** Data-driven with contractual focus
- Lead with contract language violations
- Reference specific policy sections
- Provide clear documentation timeline
- Use peer-to-peer review requests strategically

**Common Denial Reasons:**
- Medical necessity (40% of denials)
- Prior authorization missing (25%)
- Timely filing (20%)
- Network issues (15%)

**Best Response Times:** Tuesday-Thursday, 10am-2pm EST
**Escalation Path:** Appeals → Peer Review → External Review

### UnitedHealthcare
**Negotiation Approach:** Volume-based relationship building
- Emphasize provider network value
- Reference quality metrics when available
- Use batch appeals for similar cases
- Leverage provider portal data

**Key Contacts:**
- Provider Services: Focus on relationship managers
- Medical Review: Prepare comprehensive clinical documentation

**Success Strategies:**
- Bundle similar claims in appeals
- Reference Medicare reimbursement rates
- Provide comparative market analysis

### Blue Cross Blue Shield (Various Plans)
**Negotiation Approach:** State-specific strategies required
- Research state-specific regulations
- Use local medical society relationships
- Reference state insurance commission rulings
- Adapt to regional practice patterns

**State Variations:**
- **BCBS FL:** Strong on fraud prevention, detailed documentation required
- **BCBS NY:** Progressive on behavioral health, conservative on procedures
- **BCBS TX:** Volume-sensitive, relationship-based negotiations work well

### Cigna
**Negotiation Approach:** Clinical evidence-based
- Lead with peer-reviewed literature
- Provide treatment outcome data
- Use clinical pathways documentation
- Reference evidence-based guidelines

**Effective Tactics:**
- Cite specific medical literature
- Provide patient outcome data
- Use treatment algorithm documentation

## Government Payers

### Medicare
**Negotiation Approach:** Regulatory compliance focus
- Reference Medicare Coverage Database (MCD)
- Use Local Coverage Determinations (LCDs)
- Follow Medicare Claims Processing Manual guidelines
- Emphasize CMS priority conditions

**Key Resources:**
- Medicare Coverage Database
- Administrative Law Judge decisions
- Medicare Quarterly Provider Compliance newsletter

**Appeals Timeline:**
- Level 1: Medicare Administrative Contractor (120 days)
- Level 2: Qualified Independent Contractor (180 days)
- Level 3: Administrative Law Judge (90 days)
- Level 4: Medicare Appeals Council (90 days)
- Level 5: Federal District Court (60 days)

### Medicaid (State-Specific)
**Negotiation Approach:** State program compliance
- Reference state-specific Medicaid manuals
- Use state fair hearing processes
- Leverage provider shortage arguments
- Reference CMS compliance requirements

**State Strategies:**
- **CA (Medi-Cal):** Emphasize access to care issues
- **NY (Medicaid):** Use managed care contract leverage
- **FL (Medicaid):** Focus on emergency care provisions
- **TX (Medicaid):** Reference rural provider needs

## Specialty Payers

### Workers' Compensation
**Negotiation Approach:** Legal framework emphasis
- Reference state workers' comp statutes
- Use medical treatment guidelines
- Provide causation documentation
- Leverage independent medical exams

**Key Documentation:**
- First report of injury
- Medical treatment guidelines compliance
- Causation opinions
- Return-to-work documentation

**State-Specific Laws:**
- **CA:** Strict medical treatment guidelines
- **NY:** Independent medical exam requirements
- **FL:** Strong on causation documentation
- **TX:** Employer network restrictions

### Personal Injury Protection (PIP)
**Negotiation Approach:** No-fault statute compliance
- Reference state no-fault laws
- Use reasonable and necessary standards
- Provide emergency care justification
- Leverage prompt payment requirements

**Critical Elements:**
- Emergency care documentation
- Reasonable and necessary treatment
- Timely submission compliance
- State-specific benefit limits

## Payer Pattern Recognition

### High Success Rate Indicators
- Previous successful appeals with same payer
- Similar case precedents in our database
- Strong contractual language support
- Clear regulatory violation by payer

### Low Success Rate Indicators
- Repeated denials for same issue
- Payer financial distress
- Policy exclusions clearly apply
- Missing critical documentation

### Timing Optimization
- **Best Months:** January-March (new budgets)
- **Worst Months:** November-December (budget constraints)
- **Best Days:** Tuesday-Thursday
- **Best Times:** 10am-2pm local time
