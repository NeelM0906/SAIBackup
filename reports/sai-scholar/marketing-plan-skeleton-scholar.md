# Macro→Micro Marketing Plan (Legal ACT‑I Visionnaire) — Scholar Section
Date: 2026-03-03
Owner: Sai Scholar
Mission (verbatim): Sell **500** Legal ACT‑I **Visionnaire** programs at **$50K–$100K** within **90 days** (now → end of May), using **split testing on the curve of possibility** via quizzes/competitions/reports/videos/being conversations.

## 0) Non‑Negotiables (locked)
- Outcome: 500 purchases in 90 days (end of May / mid‑May summit alignment)
- Market: Lawyers (all verticals + geo segmentation)
- Operating system: no human constraint/contamination; testing within testing
- Must ship: approach + Colosseums + real-world tests + assets (LPs, videos, animations, beings, etc.)

## 1) Market Truth Brief (to be validated)
### 1.1 Core lawyer desires (hypothesis set to test)
- Win more cases (depo/cross/open/close mastery)
- Increase revenue & reduce hours (make same money, work less)
- Business growth (marketing/practice expansion)
- Certainty + control (process, repeatability)
- Reputation/status (being “best” in room)
- Exit planning (sell/step away)

### 1.2 Objection map (initial)
- Time (“I’m slammed”) → must be low-friction entry points
- Skepticism (“AI can’t understand my cases”) → must show proof + specificity
- Price resistance (“50–100k”) → must show ROI, identity, inevitability
- Fear of exposure (“I’ll be judged”) → make competitions safe/fun/aspirational

### 1.3 Competitive map (initial)
- CLE providers, trial skills courses, coaching masterminds, legal tech tools
- Differentiator: **complete holistic diagnostic + actualization** (not tips)

## 2) Message Architecture (macro narrative → micro hooks)
### 2.1 Dominant positioning (draft)
“Legal ACT‑I Visionnaire is the only complete, holistic, diagnostic, dynamic, interconnected, automated actualization system for attorneys to create more money in less time with more magic — through mastery of influence, process, and self.”

### 2.2 Pillars (draft)
1) **Diagnostic truth** (you see what you can’t see)
2) **Actualization path** (micro outcomes time‑blocked)
3) **Proof through experience** (scores/reports/competitions)

### 2.3 Entry-point hook families (for split testing)
A) **Competitions**
- “Best deposition taker” tournament
- “Best cross examiner” bracket
- “Best opening statement” showdown
- “Best closing statement” showdown
- “Best influence mastery in transactions” scenarios

B) **Quizzes**
- “What kind of agreement maker are you?”
- “Your Influence Mastery score”

C) **Reports / scorecards**
- “Your Deposition Mastery Report” (with ‘next 3 moves’)

D) **Beings / interactive**
- Talk to specialist beings (Depo Coach, Cross Examiner, Growth Architect, Exit Strategist)

E) **Video / animation**
- Micro demonstrations of diagnosis → prescription

## 3) Experiment Portfolio (curve of possibility)
### 3.1 Test matrix (Phase 1: 14 days)
- Channels: Meta / YouTube / LinkedIn / Google Search (confirm which are permitted)
- Funnels to test (parallel):
  1) Ad → quiz → report → call/app
  2) Ad → competition signup → report → being convo → call/app
  3) Ad → direct being convo → call/app
  4) Ad → webinar registration → show → app

### 3.2 Pass/Fail gates (draft)
- CTR, CPL, landing CVR, quiz completion, report unlock %, booked calls, show rate
- “Kill” underperformers fast; scale winners with variation
