# Data Provider Comparison Report
**Prepared for ACT-I | February 28, 2026**
**Requested by Mike Vesuvio**

---

## Executive Summary

Five B2B data providers were evaluated for ACT-I's automated outbound contact pipeline (Milo). The key question: **which provider(s) can deliver 20,000+ contacts/day with cell phone numbers, legal/medical vertical coverage, API access, and automation capabilities — at the best cost per contact?**

**Bottom line:** No single provider perfectly solves for all requirements. BookYourData offers the best cost-per-contact and legal/medical vertical support with free API. Apollo offers the strongest all-in-one automation platform. ZoomInfo has the deepest data but at 5-10x the cost. A hybrid approach (BookYourData for bulk data + Apollo for sequencing/automation) may be optimal.

## Recommendation for ACT-I's 20,000/Day Target

### The Challenge
Mike identified the key constraint: **"If we're capped at 20,000 a day, anything below 20,000 in utilization is a waste of money."** Most providers cap well below this:
- Apollo: 4,000/month (133/day) — **way under**
- UpLead: 400/month (13/day) — **way under**
- Seamless AI: 1,000/day shared (Pro) — **under**
- ZoomInfo: Negotiable but expensive at volume
- BookYourData: **No cap** — buy 20K credits, use them all in one day

### Recommended Hybrid Strategy

**Primary Data Source: BookYourData**
- Buy 600,000 credits/month (20K/day × 30 days) at ~$0.05-$0.08/contact = **$30K-$48K/month**
- Free API for automated ingestion
- No caps, no contracts, credits never expire
- 97% accuracy with refund guarantee
- Opt-in data (claimed) reduces TCPA exposure
- Healthcare/medical vertical lists available

**Automation Layer: Apollo.io**
- Use Organization plan ($357-$1,785/mo for 3-15 users) for sequencing engine
- Import BookYourData contacts into Apollo for email/call sequences
- Leverage Apollo's AI assistant and multi-channel workflows
- Use Apollo's 200 mobile credits/mo for priority high-value verifications

**Enrichment Supplement: ZoomInfo (optional)**
- If budget allows, ZoomInfo for high-value target enrichment only
- Use their API for deep company intelligence on qualified people
- Don't use for bulk — too expensive at volume

### Key Action Items
1. **BookYourData** — Request enterprise volume pricing for 500K-1M credits/month. Confirm legal vertical coverage (lawyer/attorney contacts specifically). Verify opt-in data claims with compliance documentation.
2. **Apollo.io** — Test Organization plan. Confirm bulk import capability from external data sources. Evaluate sequencing throughput limits.
3. **ZoomInfo** — Get quote only if deep enrichment needed. Use as supplement, not primary.
4. **TCPA Compliance** — Regardless of provider, implement phone verification layer (phone-to-name matching, line type detection) before any cell phone outreach. BookYourData's opt-in claim reduces but doesn't eliminate risk.

## Notes & Caveats
- All pricing is based on publicly available information and third-party reviews as of Feb 2026. Enterprise quotes vary significantly.
- "Opt-in" claims from BookYourData should be independently verified — most B2B data providers aggregate from public sources regardless of marketing language.
- None of these providers offer truly TCPA-safe cell phone data for cold outreach without additional consent verification.
- Legal vertical (specifically lawyer/attorney contacts) is available across all providers via industry filters, but none have a purpose-built legal database comparable to specialized legal directories.
- The 38,000 lawyer contacts already in the Seamless Google Sheet may be the strongest legal vertical asset we have — enrichment from any of these providers would add cell phones and additional data points.
