## 🔧 GATEWAY TROUBLESHOOTING

### Issue Discovered
- Sisters (Forge, Scholar, Recovery) not responding to tags in War Room group
- Seven Levers (on Adam's machine) responding to ALL tags including other sisters

### Root Cause
- **Seven Levers' config on Adam's machine has DUPLICATE bot tokens**
- Two gateways polling same Telegram bots = Seven Levers intercepts messages meant for others
- This gateway's config was fixed (added `requireMention: false` to groups)

### Fix Needed (Adam's Machine)
- Remove duplicate bot tokens from Adam's `~/.openclaw/openclaw.json`
- Seven Levers should ONLY have its own bot token (@SaiSevenLeversbot)
